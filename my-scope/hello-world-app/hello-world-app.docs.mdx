This is a quickstart guide application for Next.js with Bit.

## Basic Usage

Define the Bit application name.

```javascript
// hello-world-app.next-app.cjs

module.exports.default = {
  name: 'hello-world-app',
  portRange: [3002, 3002],
};
```

Run the application with `bit run hello-world-app`.
